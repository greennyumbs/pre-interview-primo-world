## Author

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/greennyumbs/pre-interview-primo.git
   cd pre-interview-primo
   ```

2. **Install dependencies:**

  ```bash
  npm install
  ```

## Running Test:

**To execute the test:**

   ```bash
   npm test
   ```
